# 📌 Project Overview

Human Resources teams work with large amounts of employee data every day—from hiring and employee demographics to salaries, departments, and workforce performance. However, raw spreadsheets make it difficult to identify trends, monitor workforce health, or support strategic business decisions.

This project transforms **10,000 employee records** into a fully interactive **HR Analytics Dashboard** that enables HR managers to monitor workforce metrics, analyze demographic trends, explore salary distributions, and investigate employee-level details—all from a single reporting solution.

The dashboard is designed to support both **executive-level decision making** and **operational HR analysis** through intuitive visualizations and interactive filtering.

---

# 🎯 Business Problem

Human Resource departments often face challenges such as:

- Tracking workforce growth over time
- Monitoring active, hired, and terminated employees
- Understanding employee demographics
- Identifying salary distribution across education levels
- Comparing headquarters and branch workforce
- Exploring employee records efficiently
- Finding trends hidden inside thousands of employee records

Without an interactive reporting solution, these analyses require manual effort across multiple spreadsheets, making decision-making slower and less reliable.

The dashboard dynamically calculates metrics such as:

- 👨‍💼 Total Hired Employees

- ❌ Terminated Employees
- 👥 Employee Count by Department
- 🌍 Headquarters vs Branch Employees

# 🔄 Dashboard Actions

To create an application-like experience, multiple dashboard actions were implemented.

Users can simply click on a visualization to instantly filter related charts.

Examples include:

- Clicking a department filters demographic and salary charts.
- Selecting a gender updates all workforce metrics.
- Choosing an education level filters salary analysis.
- Selecting a state updates employee distribution.

This minimizes manual filtering and improves exploratory analysis.

# 🧠 Challenges Faced

Developing this dashboard involved solving several real-world Tableau challenges.

### Challenge 1 — Creating Dynamic HR KPIs

**Problem**

Business KPIs needed to update automatically whenever filters changed.

**Solution**

Developed reusable calculated fields that dynamically recalculate all HR metrics.

---

### Challenge 2 — Dashboard Navigation

**Problem**

Users needed an intuitive way to move between dashboards without opening worksheets manually.

**Solution**

Implemented navigation buttons to create an application-like user experience.

---

### Challenge 3 — Dashboard Space Optimization

**Problem**

Displaying all filters permanently reduced the available dashboard space.

**Solution**

Built a collapsible Show/Hide Filter Panel, allowing users to access filters only when required.

---

### Challenge 4 — Rich User Experience

**Problem**

Default Tableau tooltips provide limited context.

**Solution**

Designed custom tooltip worksheets that display additional information and visual context on hover.
